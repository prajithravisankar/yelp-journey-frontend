# Yelp Journey – Frontend Notes

## 0. Stack & Setup

- Framework: Next.js 15 (App Router) + React + TypeScript
- Styling: Tailwind CSS + shadcn/ui
- Animation: Framer Motion
- Data fetching: React Query
- Maps: Mapbox GL JS
- Icons: Lucide
